# Power_BI_Dashboard_Project
📊 Mobile Sales Power BI Dashboard
📌 Project Description

The Mobile Sales Power BI Dashboard is used to analyze and visualize mobile phone sales data. The dashboard provides insights about sales performance, top-selling brands, revenue trends, and regional sales distribution using interactive charts and visualizations.

This project demonstrates how Power BI can transform raw sales data into meaningful business insights.

🎯 Objectives

1.Analyze mobile phone sales performance.

2.Identify top-selling mobile brands and models.

3.Track monthly sales trends.

4.Compare sales across different cities or regions.

5.Create interactive visualizations for better decision-making.

🛠 Tools & Technologies

1.Power BI Desktop

2.Microsoft Excel / CSV

3.Power Query

4.DAX (Data Analysis Expressions)

📂 Dataset Information

The dataset contains mobile sales transaction records.

Dataset Columns
Column Name	    Description
Order_ID	    Unique order number
Date	        Date of purchase
Brand	        Mobile brand
Model	        Mobile model
Price	        Price of mobile phone
Quantity	    Units sold
Total_Sales	    Total revenue
City	        Sales location
Customer_Name	Name of customer

📈 Dashboard Visualizations

The Power BI dashboard includes:

1.Total Sales KPI Card

2.Sales by Brand (Bar Chart)

3.Monthly Sales Trend (Line Chart)

4.Top Selling Models

5.Sales by City (Map or Bar Chart)

6.Interactive Filters / Slicers

📊 Key Insights

Using this dashboard we can analyze:

1.Highest selling mobile brand

2.Most popular mobile models

3.Total revenue generated

4.City-wise sales distribution

5.Monthly sales growth

▶ How to Use

1.Open Power BI Desktop.

2.Import the dataset (Excel / CSV file).

3.Create relationships and measures if required.

4.Build visualizations using charts and slicers.

5.Analyze mobile sales performance.

📌 Conclusion

The Mobile Sales Power BI Dashboard helps businesses track sales performance, identify trends, and make data-driven decisions using interactive data visualization.
